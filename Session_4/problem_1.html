Problem 1

There have been a bunch of cats and dogs which got loose recently! Luckily, they all have tags which list their owners' names. Furthermore, being the good owners they are, a bunch of people have called into the shelter asking to be contacted if their pets show up, but not everyone's pet has shown up yet. You're in charge of making sure that each owner gets notified that their pet has arrived , and as an aspiring software engineer, you've decided to write a program to do the work for you!

You're going to be filling in code inside the shelter() function. The function takes in two arguments: an ARRAY of people's names, and a DICTIONARY of pets' names and their owners. The function should return an ARRAY of people whose pets are at the shelter!

Example

Dictionary of owners who have called in: owners = {"Bob" : "Fluffy", "Joanne" : "Spike", "Alex" : "Fido"}
Array of pets at the shelter: pets = ["Fluffy", "Spike"]

Calling the shelter function: shelter(owners, pets) --> ["Bob", "Joanne"]
